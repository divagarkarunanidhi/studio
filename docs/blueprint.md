# **App Name**: Defect Insights Dashboard

## Core Features:

- CSV Data Upload: Allow users to upload a CSV file containing defect data.
- Defect Creation Stats: Display the total number of defects created yesterday.
- Grouped Defect Analysis: Group defects based on user input, such as severity, priority, or assigned user.
- Ready for Testing Status: Show a count of defects currently marked as 'Ready for Testing'.
- Domain Categorization: Automatically categorize defects based on their domain from the data provided in CSV.
- Defect Origin: List of people who created and/or reported each defect, automatically extracted from the CSV file.
- Urgent Defect Flagging: Flag defects in 'To-Do' status that were created more than a week ago by highlighting them in red.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of trust and reliability.
- Background color: Light Grey (#F0F2F5), for a clean and professional look.
- Accent color: Soft Violet (#9575CD), providing a touch of uniqueness while remaining subtle.
- Body and headline font: 'Inter', sans-serif, known for its clean and readable design, ensuring clear presentation of data.
- Use simple, line-based icons for representing defect status, domains, and actions, promoting easy recognition.
- Dashboard layout with clear sections for each menu option, utilizing card-based design for organizing data.
- Subtle transitions and loading animations to enhance user experience without being distracting.
